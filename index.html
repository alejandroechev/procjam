Cristian Donoso and Alejandro Echeverria 2014.

Archipelago Generator

A procedural tool for creating 2D maps of archipelagos.

You can modify the number of islands, number of rows or columns and press update to generate the new map. Yo can apply a post-processing kernel to the generate result grid to increase the size of the islands. You can also add new procedural islands by clicking anywhere on the map.

A text version of the map will be continually updated on the text area in the lower right corner of the page, which you can use to export the generated archipelago.
